# shopping_cart

A production-ready e-commerce shopping cart built with Flutter and Dart. This project implements Clean Architecture (Data, Domain, and Presentation layers) and the BLoC pattern for predictable state management. It features Local Storage for cart persistence, ensuring a seamless user experience even after app restarts.
